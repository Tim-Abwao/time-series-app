Uploading files

A quick word about uploads...

  • Please upload files in CSV format
  • Files should be no larger than 15MB
  • The data to be processed should be in the last column
  • The data should have at least 21 rows
  • The index should consist of dates, in standard fromats.

For example,

Index(Date) ... Data
2020-01-01 ... 140.25
2020-01-02 ... 187.50
... ... ...
